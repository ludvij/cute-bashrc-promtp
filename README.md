# cute-bashrc-promtp
A cute (imo) bash and powerhsell prompt
## How to use
### bash
 - source it in your .bashrc file
 - copy it to your .bashrc file
### powershell
 - copy it to your ps1 profile
 - run it from your ps1 profile  
⚠️you may need some .NET libraries but I'm not sure
